Marquee Type — Chalet / 235px / inherited line height / 20px letter spacing

Client 

<p class="marquee-container">
    <span class="marquee-container__marquee-type" style="animation: marquee 5s linear infinite;">Client</span>
</p>

.marquee-container {
    width: 100%;
    height: 275px;
    white-space: nowrap;
    overflow: hidden;
}
.marquee-container__marquee-type {
    font-family: $chalet;
    font-size: 235px;
    line-height: inherit;
    letter-spacing: 20px;
    display: inline-block;
    padding-left: 100%;
    margin-top: 120px;
}

Note that the animation property is an inline style. The animation duration should be content managed because depending on how long or short the client name is, the text will appear to be moving faster or slower.
